AI 翻译助手 - 使用指南

如何获取 API Key

第一步：注册阿里云账号

访问 阿里云百炼平台 并注册/登录。

第二步：开通百炼服务

进入控制台开通模型服务，新用户有免费额度。

第三步：获取 API Key

左侧菜单「API Key 管理」创建并复制。

第四步：填入插件

点击插件图标 > 设置 > 粘贴 API Key > 保存。

功能说明

▶
划词翻译 - 选中文字自动翻译，单词给简明释义，段落翻译全文。
▶
详细讲解 - 点击「详细讲解」获取词性、释义、例句、用法。
▶
提取难词 - 段落翻译后提取关键词，可逐个查看详解。
▶
整页翻译 - 一键翻译页面（含 iframe），可还原。
▶
上下文分析 - 自动捕获上下文提供更准确释义。
▶
流式输出 - 翻译结果逐字显示，无需等待。
▶
动态内容监听 - 整页翻译后自动监听页面新加载的内容并翻译，适合无限滚动页面。
▶
在线文档模式 - 针对 Google Docs、Office Online 等无法划词翻译的在线文档，开启后复制内容即自动翻译，结果显示在右下角浮窗。
▶
历史记录 - 自动保存，支持收藏/删除/导出/导入。

推荐模型

qwen-turbo - 速度快、价格低，推荐日常使用

qwen-plus - 质量更好的均衡选择

qwen-max - 最高质量，适合复杂内容

已知限制

Google Docs、Office Online 等 Canvas 渲染的文档无法划词翻译，但可以使用「在线文档模式」：开启后复制文字即自动翻译。

常见问题

翻译没反应？ - 检查 API Key 和账户余额。

模型不存在？ - 检查模型名称拼写。

更新后报错？ - 刷新页面即可。
